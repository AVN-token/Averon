AVN — Averon

AVN (Averon) is a Polygon-based utility token designed to support sustainable ecosystem growth, liquidity stability, and long-term development through a transparent and deflationary model.

Overview
Token Name: Averon Network
Symbol: AVN
Network: Polygon
Total Supply: 100,000,000 AVN
Token Type: Utility Token

AVN has a fixed total supply with no mint function, ensuring long-term supply transparency.

Deflationary Mechanism
30% of the total supply (30,000,000 AVN) has been permanently burned.
The burned tokens are sent to a dead address and are irreversibly removed from circulation.

Tokenomics
